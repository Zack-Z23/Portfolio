HTML

Getting started with HTML and CSS. Html is a programming language that allows you to code websites of your choosing. Websites you can create can range from educational to websites that allow you to download stuff or maybe it is a bank you are making a website on. To customize HTML websites you need CSS which will allow you to change characteristics of specific elements in the HTML code.

With HTML some syntaxes allow you to add text to a website and some others allow you to add images onto the website. Html tags allow the user to add content to the website and there are certain types of tags that will get you different things. An HTML element is the root of the whole website and what tells your VScode what type of document it is. Attributes in HTML are what specifies a path to content you want in your “” tag or “” tag. Then inside HTML, there are different elements types in two categories, inline and block.

As the image explains above, the inline elements are meant to be elements that go side by side with each other whilst block elements are stacked on top of each other vertically. Most websites that are made by professionals use block and inline elements on their website to align content and make the content easier to read.

Moving on to CSS. CSS is basically what styles everything and makes it look nice and aligned. In CSS it allows you to put null space between each element and whitespace when needed. CSS’s main purpose is to style content on HTML and get that definition out the way, the CSS rule is when you group more or one CSS property that styles more or one HTML element. A CSS selector selects an HTML element and styles it, hence the name selector.

H1 and P are both selectors because they select specific HTML elements.

In this case. Yellow is the value because you are trying to put the background color of an anchor to yellow. In another case, if you were trying to remove the underline on a link you would put,

A{ text-decoration: none; }

This makes sure that there won’t be any decorations on the text so there would be no underlines on your links
